    Welcome to SM (Screen Message)

    This is Web edition of screen-message, we can use it with Firefox, Chrome. and it`s not support Internet Explorer.


    HowTo

    We can click the link or key in something with browser.

    # open the source web link.
firefox http://sm.nomeata.de/

# or you can open the mirror link on GitHub.
firefox http://sm.drx.tw/sm.html

# now, we can input color values at link (with javascript).
firefox http://sm.nomeata.de/sm.html#t=%3A-);b=#000;f=red

    Color Scheme

    • Default - system default backgound and color. [mirror]
    • Dark & Red - dark backgound and red color. [mirror]
    • Dark & Green - dark backgound and Green color. [mirror]
